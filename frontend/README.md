Automated Student Attendance Monitoring and Analytics System

Overview
This project is a smart attendance management system designed for colleges. It uses facial recognition technology to automatically detect students and record their attendance. The system also provides analytics and reports that help teachers and administrators monitor attendance patterns and identify defaulter students.

Features Face recognition-based attendance marking Student, Teacher, and Admin dashboards Automated attendance recording Attendance analytics (daily, weekly, monthly) Defaulter list for students below 75% attendance Centralized attendance database Easy student management for teachers and admins System Modules

Student Panel Mark attendance using Face ID View attendance records and percentage
Teacher Panel View student attendance data Add or remove students Generate attendance reports View defaulter students lis
Admin Panel Manage teachers and students Monitor system data Access full attendance analytics
Technologies Used Frontend: React, HTML, CSS, JavaScript
Backend: Node.js, Express.js Database: MongoDB Face Recognition: OpenCV (Python)

Working Process Student opens attendance system Camera captures face image System detects and recognizes face Face matched with database Attendance marked automatically Data stored in database Reports and analytics generated
